除了最近很火的 spectral norm 之外，pytorch 还实现了 2016 年 OpenAI 的文章：Weight Normalization: A Simple Reparameterization to Accelerate Training of Deep Neural Networks。该模块用于归一化网络权重，从而加速训练。本文介绍该模块的原理和 pytorch 的实现细节。

Weight Normalization

通常的梯度下降法直接将损失函数对 weight 求导得到梯度，然后以一定学习率，沿着梯度下降的方向更新权重。而 weight normalization 将每个神经元 weight 的方向和长度解耦，即把参数 weight 分为两部分：v 和 g，其中 v 代表方向，而 g 代表长度。然后将损失函数分别对这两部分求导，并更新它们的值。文章通过实验和理论分析，证明这种方式可以加速网络收敛。

(9) 最后一行，中间括号部分为一个投影矩阵，将空间中任意一个向量投影到向量 w 的补空间（垂直于 w 的超平面）。经过 weight normalization 后，损失函数对 v 的梯度，其实是它在 w 的梯度基础上，做了两个方面的改进：

  • 只保留垂直于 w 方向的部分
  • 长度放缩为 g/||v||

由于 v 和 w 的方向相同，而 垂直于 v ，根据勾股定理，随着迭代次数的增加，v 的模是单调递增的。当 很大时，经过若干次迭代，||v|| 会很快增大，导致 g/||v|| 减小，从而降低 ；而当 很小时，||v|| 会停止进一步增大。因此，weight normalization 可以使梯度值更稳定。

梯度前面的放缩项 g/||v||，可以使 learning rate 的选择范围更广，当 learning rate 很大时，||v|| 会迅速增大从而让梯度的放缩项减小，稳定训练。

Pytorch Implementation

pytorch 的官方实现代码为：https://github.com/pytorch/pytorch/blob/master/torch/nn/utils/weight_norm.py

可以直接无脑调用：

from torch.nn.utils import weight_norm
import torch

conv = weight_norm(torch.nn.Conv2d(3, 10, 5),name='weight')
fc = weight_norm(torch.nn.Linear(20, 40), name='weight')

conv.weight_g.shape # 卷积核的输出包括10个layer，这 10 个值对应原始 weight 的 10 个 norm
# torch.Size([10, 1, 1, 1])
conv.weight_v.shape # 和原始的卷积核形状相同
# torch.Size([10, 3, 5, 5])

看一下 pytorch 官方的实现细节：

from torch.nn.parameter import Parameter
from torch import _weight_norm, norm_except_dim


class WeightNorm(object):
    def __init__(self, name, dim):
        if dim is None:
            dim = -1
        self.name = name # 一般都是 weight
        self.dim = dim # 默认是 0，即输出的卷积层数、neuron 数，在weight中所在的维度

    def compute_weight(self, module):
        # 这个函数用于从 weight_g 和 weight_v 反推 weight
        # 公式即为：weight = weight_g*weight_v/|weight_v|
        g = getattr(module, self.name + '_g')
        v = getattr(module, self.name + '_v')
        return _weight_norm(v, g, self.dim)

    @staticmethod
    def apply(module, name, dim):
        for k, hook in module._forward_pre_hooks.items():
            if isinstance(hook, WeightNorm) and hook.name == name:
                raise RuntimeError("Cannot register two weight_norm hooks on "
                                   "the same parameter {}".format(name))

        if dim is None:
            dim = -1

        fn = WeightNorm(name, dim)

        weight = getattr(module, name) # 获取原始权重

        # remove w from parameter list
        # 从模型的参数列表中删除 weight
        del module._parameters[name]

        # add g and v as new parameters and express w as g/||v|| * v
        # 初始化参数 weight_g 和 weight_v, 其中 weight_g 保存 neuron 的权重，
        # weight_v 初始化为原始的 weight
        module.register_parameter(name + '_g', Parameter(norm_except_dim(weight, 2, dim).data))
        module.register_parameter(name + '_v', Parameter(weight.data))
        # 通过 fn 调用本类中的函数 compute_weight，从 weight_g 和 weight_v 重新计算 weight
        # 并设置为新的成员变量 weight，注意这里的 weight 仅仅是成员变量，不再是网络的参数了，
        # 不会被直接通过梯度下降法更新 
        setattr(module, name, fn.compute_weight(module)) 

        # recompute weight before every forward()
        # 设置前向传播的钩子，fn 默认调用本类中的 __call__ 方法
        module.register_forward_pre_hook(fn)

        return fn

    def remove(self, module):
        # apply 的逆操作，重新计算 weigh
        weight = self.compute_weight(module)
        # 删除 weight 成员变量
        delattr(module, self.name)
        # 从计算图中删除 weight_g 和 weight_v
        del module._parameters[self.name + '_g']
        del module._parameters[self.name + '_v']
        # 将 weight 添加回参数列表
        module.register_parameter(self.name, Parameter(weight.data))

    def __call__(self, module, inputs):
        # 用于设置前向传播钩子，在运算之前，先将该模块成员变量 weight 用 weight_g 和 weight_v 更新
        setattr(module, self.name, self.compute_weight(module))

def weight_norm(module, name='weight', dim=0):
    # 通过 WeightNorm.apply 的静态方法，修改 module 内部的参数机制，共做了如下操作：
    # 1) 删除了 module 中的 weight parameter，将其修改为普通的成员变量（即不直接参与权重更新） 
    # 2) 初始化了模块新的 named_parameter：weight_g 和 weight_v,用它们取代之前的 weight
    # 3) 由于模块前向传播相关代码中，没有对 weight_g 和 weight_v 的操作，仍然利用 weight,
    #    为了避免牵一发动全身，重新设置 模块的成员变量 weight，用 weight_g 和 weight_v 计算得到
    WeightNorm.apply(module, name, dim)
    return module
